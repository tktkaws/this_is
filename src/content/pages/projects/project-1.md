## 概要

大規模 EC サイトのフロントエンドを React + Next.js でフルリニューアルしたプロジェクトです。

## 技術スタック

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- GraphQL

## 担当範囲

- フロントエンドアーキテクチャの設計
- コンポーネント設計・実装
- パフォーマンス最適化

## 成果

- ページ読み込み速度を 40% 改善
- Lighthouse スコア 90 以上を達成
